### TechTool

Инструмент, постороеный на базе Raspberry Pi, для контроля электрооборудования автомобилей. Подключается через разъем ODB2 и использует бортовое питание. В качестве дисплея предполагается использование смартфона, таблета или ноутбука.
Доступ производится подключением по WiFi (Raspberry Pi выступает в роли точки доступа). Коммуникация через Web-интерфейс или по терминалу ssh-соединения.


### Image

Имидж можно найти в подкаталоге img. Записывается на флешку от 4GB. Основан на последней версии Raspberry Pi OS 64.
